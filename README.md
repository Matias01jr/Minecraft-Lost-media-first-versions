# Como instalarlos
## Copia las carpetas en los directorios correspondientes:
### En Windows es: Ejecutar > %appdata% > ver > mostrar archivos ocultos
.minecraft/versions

### En Linux es: Abrir gestor de archivos (thunar, pcmanfm, etc) > ver > mostrar archivos ocultos
.minecraft/versions

### Una vez llegado al directorio de tu lanzador pega la carpeta entera, luego busca en tu lanzador y ejecuta
Si no funciona, prueba en una máquina con Linux, o puedes revisar la estructura de otras versiones instaladas y ver qué archivo faltan y copiarlos manualmente
