# English

# How to Install
## Copy the folders to the corresponding directories:
### On Windows: Run > %appdata% > View > Show hidden files
.minecraft/versions

### On Linux: Open your file manager (Thunar, PCManFM, etc.) > View > Show hidden files
.minecraft/versions

### Once you're in your launcher's directory, paste the entire folder, then search for it in your launcher and run it.
If it doesn't work, try on a Linux machine, or you can check the structure of other installed versions and see which files are missing and copy them manually.
